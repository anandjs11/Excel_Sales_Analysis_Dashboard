**Superstore Sales Analysis (Excel Dashboard)**

*Project Overview*

This project analyses Superstore sales data (1,000 transactions) to identify trends, loss-making products, and the impact of discounts on profitability. The analysis was carried out using Microsoft Excel, and a simple dashboard was created to summarise key insights.

**_Work Done_**
1. Data Cleaning

Converted raw CSV into a structured Excel sheet

Verified data types for dates and numerical fields

Prepared the dataset for analysis and pivot tables

2. Feature Engineering

Added new calculated columns to support analysis:

Discount grouping / ranges

Flags to identify loss-making transactions and high-discount cases

3. Analysis

Identified loss-making products

Calculated profit and profit margin by product

Analysed the relationship between discount levels and profit margin

Found the top loss-making products

4. Pivot Tables

Created pivot tables to analyse:

Profit by product

Profit by category and sub-category

Discount vs profit margin

Top loss-making products

5. Dashboard

Built a simple Excel dashboard to visualise:

Top loss-making products

Profit by category

Discount vs margin trends

Added filters for easier exploration

**Key Insights**

High discounts are often associated with negative profit margins

Some products consistently generate losses and require pricing or discount review

Profitability varies significantly even within the same category
